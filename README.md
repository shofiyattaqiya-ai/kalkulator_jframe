# kalkulator_jframe
aplikasi kalkulator sederhana yang dibuat menggunakan jframe
program ini mampu melakukan operasi aritmatika dasar melalui graphic user interface (GUI) yang mudah digunakan

## fitur 
1 penjumlahan
2 penguragan
3 perkalian
4 pembagian
5 dan sebagainya

## teknologi
1 java
2 java swing (Jframe)
3 net beans IDE


## cara menjalankan
1 clone repository
'''bash
git clone https://github.com/shofiyattaqiya/kalkulator_jframe.git
'''
2 buka project menggunakan net beans IDE
3 run/jalankan aplikasi kalkulator_Jframe
